🔧 Manual Configuration

If automatic updates aren't working, you can manually import the configuration:

  1. Open the MahalleChat app
  2. Go to Settings → Configuration
  3. Tap "Import Configuration"
  4. Visit config.json
  5. Copy the entire JSON content
  6. Paste into the app and tap "Import"
